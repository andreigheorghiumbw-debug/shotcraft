# 🚀 ShotCraft
    
Transform messy screenshots into beautiful marketing assets instantly.

Stop wasting 30 minutes in Figma just to share a progress update. ShotCraft helps you turn raw screenshots into professional-grade marketing assets in seconds.

## ✨ Features
- **Full-page capture using chrome.tabs.captureVisibleTab**
- **Bento-box layout generator (HTML/CSS)**
- **Background gradient picker**
- **One-click export to PNG**
- **☕ Support the Creator:** Built by Andrei (@aer.solutions)

## 🛠 Installation
1. Download this folder.
2. Go to `chrome://extensions/` in Chrome.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select this folder.
